# 🏗️ أركان القوة - موقع شركة مقاولات وإنشاءات

موقع ويب حديث وفاخر لشركة **أركان القوة** المتخصصة في المقاولات والإنشاءات العامة.

## 📋 نبذة عن المشروع

هذا الموقع يمثل الواجهة الرقمية الاحترافية لشركة مقاولات رائدة تجمع بين:
- ✨ عراقة الخبرة الهندسية
- 🎨 حداثة التصميم والتقنيات الحديثة
- 🌐 تطبيقات ثلاثية الأبعاد (3D) متقدمة
- 📱 تجربة مستخدم متجاوبة (Responsive Design)

### الخدمات الرئيسية:
- 🏢 المشاريع السكنية
- 🏪 المشاريع التجارية
- 🏭 المشاريع الصناعية
- 🏗️ البنية التحتية الحكومية
- 📐 التصاميم الهندسية ثنائية وثلاثية الأبعاد

## 🛠️ التقنيات المستخدمة

### Frontend Technologies:
| التقنية | الوصف |
|--------|------|
| **HTML5** | لغة ترميز قياسية حديثة |
| **CSS3** | تنسيق متقدم مع متغيرات CSS و Gradients |
| **JavaScript (ES6+)** | برمجة ديناميكية وتفاعلية |

| **GSAP** | مكتبة الرسوم المتحركة المتقدمة (GreenSock Animation Platform) |
| **AOS (Animate On Scroll)** | تأثيرات الحركة عند التمرير |
| **Leaflet.js** | مكتبة الخرائط التفاعلية |
| **FontAwesome 6** | مجموعة أيقونات عصرية |
| **Google Fonts** | خطوط عربية احترافية (Cairo, Tajawal) |

## 🚀 البدء السريع

### 1. تحميل الملفات
تأكد من أن لديك جميع ملفات المشروع:
- `index.html`
- `css/style.css`
- `js/main.js`
- `js/three-app.js`
- مجلد `images/`

### 2. فتح الموقع
**الطريقة الأولى - فتح مباشر:**
- انقر مزدوجاً على ملف `index.html`

**الطريقة الثانية - باستخدام خادم محلي (مُفضَّل):**
```bash
# استخدام Python 3
python -m http.server 8000

# أو استخدام Node.js
npx http-server

# أو استخدام Live Server في VS Code
# ثم افتح: http://localhost:5500
```

### 3. الوصول إلى الموقع
```
http://localhost:8000
أو
http://localhost:5500
```

## ✨ المميزات الرئيسية

### 🎯 الواجهة الأمامية (Header & Navigation)
- شريط تنقل عائم ثابت مع تأثيرات حركة ناعمة
- شعار الشركة مع تأثير متدرج لوني
- قائمة ملاحة كاملة بجميع الأقسام
- زر "طلب عرض سعر" بارز

### 🌟 قسم البطل (Hero Section)
- صورة خلفية جذابة مع تدرج لوني فاخر
- محتوى نصي ملهم ودعوة للعمل واضحة
- تصور ثلاثي الأبعاد تفاعلي (Three.js)
- حركات الدخول السلسة (GSAP)

### 📊 قسم الإحصائيات
- عرض أرقام مؤثرة للشركة (سنوات الخبرة، المشاريع المُنجزة، الموظفون، العملاء الراضون)
- رسوم متحركة الأرقام

### 🏢 قسم من نحن (About)
- سرد مقنع عن تاريخ الشركة
- رؤية وأهداف الشركة
- نقاط القوة والمميزات

### 💼 قسم الخدمات (Services)
- عرض تفاعلي للخدمات المختلفة
- تأثير الإمالة ثلاثية الأبعاد عند تمرير الفأرة
- وصف تفصيلي لكل خدمة

### 🎨 قسم المشاريع (Portfolio)
- معرض مشاريع سابقة ناجحة
- صور عالية الجودة
- تفاصيل كل مشروع

### 📝 المدونة (Blog)
- مقالات مفيدة وتحديثات الشركة
- محتوى غني بالمعلومات

### 💼 قسم الوظائف (Careers)
- عرض فرص العمل المتاحة
- معلومات عن الشركة كمكان عمل

### 📍 خريطة وتفاصيل التواصل
- خريطة تفاعلية (Leaflet)
- معلومات الاتصال الكاملة
- نموذج التواصل

## ⚙️ المكتبات الخارجية

جميع المكتبات يتم تحميلها من CDN عام:

```html
<!-- Bootstrap 5 RTL -->
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.2/dist/css/bootstrap.rtl.min.css">

<!-- FontAwesome 6 Icons -->
<link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/font-awesome/6.4.2/css/all.min.css">

<!-- AOS - Animate On Scroll -->
<link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/aos/2.3.4/aos.css">

<!-- Leaflet Maps -->
<link rel="stylesheet" href="https://unpkg.com/leaflet@1.9.4/dist/leaflet.css">

<!-- Google Fonts - Cairo & Tajawal -->
<link href="https://fonts.googleapis.com/css2?family=Cairo:wght@300;400;500;600;700;800;900&family=Tajawal:wght@300;400;500;700;900&display=swap" rel="stylesheet">

<!-- GSAP Animation Library -->
<script src="https://cdnjs.cloudflare.com/ajax/libs/gsap/3.12.2/gsap.min.js"></script>

<!-- Three.js -->
<script src="https://cdnjs.cloudflare.com/ajax/libs/three.js/r128/three.min.js"></script>
```

## 🔄 التطوير والتخصيص

